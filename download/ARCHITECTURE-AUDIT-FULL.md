                ↓           ↑

## B. INPUT PIPELINE

### B.1 Keyboard Pipeline

```
Hardware key press → window keydown event
    ↓
InputSystem.onKeyDown(e):
    Space    → kbEdge.jump=true, callbacks.jump?.() if !gameplayBlocked
    KeyJ     → kbEdge.fire=true, callbacks.fire?.()
    KeyK     → kbEdge.melee=true, callbacks.melee?.()
    KeyE     → kbEdge.interact=true, callbacks.interact?.()
    KeyQ     → kbEdge.weaponPrev=true, callbacks.weaponPrev?.()
    KeyF     → kbEdge.grapple=true, callbacks.grapple?.()
    KeyG     → kbEdge.emp=true, callbacks.emp?.()
    Escape   → kbEdge.pause=true AND kbEdge.back=true, callbacks.pause?.()
    Shift    → kbEdge.dash=true, callbacks.dash?.(dir)
    WASD     → state.held{Left,Right,Up,Down} = true

[PER FRAME] InputSystem.update():
    Merges kbEdge + gamepad → state (mixed flags + gp* flags)
    Clears kbEdge buffer
```

### B.2 Gamepad Pipeline

```
[PER FRAME] InputSystem.update():
    Polls navigator.getGamepads()
    Button edges: edge(0)=A/jump, edge(2/7)=RT/fire, edge(3)=Y/melee,
                  edge(1)=B/interact+back, edge(6)=LT/dash,
                  edge(5/4)=R1/L1/weaponSwitch, edge(9)=Start/pause,
                  edge(12/13)=D-pad Up/Down (grapple/emp + heldUp/heldDown),
                  held(14/15)=D-pad Left/Right
    Sets gp* flags (gamepad-only, B2 fix)
    Sets state.held* flags
```

### B.3 Mouse Pipeline

```
UIController.addButton() → bg.setInteractive() + pointerover/pointerdown handlers
    pointerover → focusIndex update + uiHover sound
    pointerdown → uiClick sound + onSelect()

[CURSOR MODE] (right stick):
    processCursorHover() → hitTest with mockPointer → lastHovered
    A button → lastHovered.emit('pointerdown')
```

### B.4 Gates

| Gate | Effect |
|------|--------|
| gameplayBlocked | Blocks gameplay callbacks in onKeyDown |
| OverlayManager.hasOpen | Redirects all input to overlay |
| this.paused | Freezes updatePlay, routes to PauseMenuUI |
| navCooldown (120ms) | Rate-limits UI navigation |
| clickCooldown (200ms) | Rate-limits cursor clicks |
| lastPauseToggleAt (200ms) | Debounces pause toggle |

## C. UI SYSTEM ARCHITECTURE

### C.1 UIController

**Two modes (auto-detected):**
1. **Focus mode** — D-pad/stick/arrow keys navigate between buttons
2. **Cursor mode** — right stick moves virtual cursor, A simulates click

**Key methods:**
- `addButton(x, y, bg, onSelect, opts)` — registers focusable + wires mouse handlers
- `clearFocusables()` — removes all listeners (S1 fix: NO removeInteractive)
- `findNearest(direction)` — 2D spatial navigation (up/down/left/right)
- `addTabs(tabs)` — for L1/R1 tab switching
- `setupKeyboard()` — window keydown listener for Enter/Space/arrows

### C.2 OverlayManager

Stack-based overlay management:
- `open(id, ui, parent)` → push + show
- `close(onClose)` → pop + hide + destroy
- `handleInput(onClose)` → ESC/B closes, else delegates to ctrl.update()

## D. GAMEPLAY SYSTEMS

### D.1 PlayerEntity (1090 lines)

**Stats computation:** base × skillMult × chassisMult
**Movement:** Matter physics, coyote time, jump buffer, wall slide, dash, double jump
**Combat:** fire (projectile/hitscan/explosive), melee (200ms commit), 8 weapons
**Abilities:** hover (hold jump), grapple (F), EMP (G), hack (hold E)
**Animation commitment:** FIRE_COMMIT_MS=80, MELEE_COMMIT_MS=200

### D.2 Enemy System

6 types: drone, spider, heavy, sniper, flying_ai, elite
FSM: patrol → aggro → attack (telegraph→window→recovery) → stagger
Posture system (Souls-like): fills on hit, stagger when full, 50% bonus damage window

### D.3 Boss System

2 bosses: guardian_ax09 (Act I), neural_overseer (Act IV)
Phased AI (2 phases), 4 attack types: shoot, lunge, teleport, beam
Death sequence: kneel animation → BOSS_DEAD event → victory

### D.4 Collision System

Single collisionstart listener, routes by getData() attributes:
section triggers, checkpoints, boss entry, enemy contact, hazards

### D.5 Metroidvania System

Collectibles (HP/energy/skill/weapon), shortcuts, EMP doors, grapple anchors
Per-frame distance checks (35px collectibles, 60px shortcuts)
