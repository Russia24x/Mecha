# MECHA: LAST PROTOCOL
## Design Pillars Bible v1.0

> **The world is not waiting to be saved. It is waiting to be understood.**

> **Nothing exists only because it looks cool. Everything must exist because the world needs it.**

---

## Combat

```
Heavy
Precise
Punishing
```

هر ضربه وزن دارد. هر شلیک عاقبت دارد. مبارزه رقص نیست — تصمیم است.

- **Heavy:** هر حرکت تعهد است. نمی‌توان mid-animation لغو کرد. ضربه‌ای که شروع می‌شود، تمام می‌شود.
- **Precise:** hitboxها دقیق‌اند. telegraph واضح است. اگر زدم، تقصیر بازیکن است.
- **Punishing:** اشتباه = آسیب. آسیب = مرگ سریع. اما عادلانه. همیشه راه فرار هست.

**نجات:** دشمن تهدید است، نه مزاحم. هر fight باید حس "تقریباً مردم" بدهد.

---

## Exploration

```
Curiosity
Reward
Secrets
```

هر کنج好奇心 باید پاداش داشته باشد. هر پاداش باید کنجکاوی بیشتر بسازد.

- **Curiosity:** بازیکن باید ببیند "اون اون طرف چیست؟" و بخواهد برود.
- **Reward:** پاداش همیشه مادی نیست. گاهی لور است. گاهی میان‌بر. گاهی فقط زیبایی.
- **Secrets:** هیچ منطقه‌ای نباید بدون حداقل یک مخفی باشد. حتی اگر کوچک.

**نجات:** اگر بازیکن ۵ دقیقه در یک направление می‌رود و چیزی پیدا نمی‌کند، طراحی شکست خورده.

---

## Lore

```
Environmental
Minimal Dialogue
Player Interpretation
```

داستان از طریق محیط روایت می‌شود. دیالوگ کم است. بازیکن خودش معنا می‌سازد.

- **Environmental:** لاشه مکاها، پیام‌های قدیمی، ساختمان‌های ویران — همه داستان می‌گویند.
- **Minimal Dialogue:** NPCها کم حرف می‌زنند. وقتی حرف می‌زنند، مهم است.
- **Player Interpretation:** هیچ cutscene‌ای داستان را "توضیح" نمی‌دهد. بازیکن خودش کنار هم می‌گذارد.

**نجات:** اگر بازیکن بعد از ۳ ساعت نگوید "فکر کنم فهمیدم..."، لور شکست خورده.

---

## Boss

```
Every boss teaches something.
```

هر باس یک **ایده** است. بازیکن بعد از شکست دادنش احساس می‌کند با یک فلسفه جنگیده است.

- هر باس یک جمله دارد (شعار)
- هر باس یک ایده دارد (مفهوم فلسفی)
- هر باس یک خاطره می‌سازد (احساس خاص)
- هر باس مکانیک مبارزه‌اش با ایده‌اش هماهنگ است

| Boss | ایده | مکانیک |
|------|------|--------|
| Guardian AX-09 | Duty without purpose | مقاوم، کند، حمله‌های قابل پیش‌بینی — نمی‌خواهد بجنگد |
| Leviathan Hulk | Sacrifice without witness | بزرگ، روی بدنه‌اش می‌جنگی، حمایت می‌کند از چیزی که نیست |
| Iron Magistrate | Justice without mercy | متعادل، دقیق، بی‌رحم — هر اشتباه جریمه سنگین |
| The Gardener | Mercy without consent | دردناک چون "نجات" می‌دهد — ریشه‌ها محدودت می‌کنند |
| The Architect | Perfection without freedom | تکامل‌یافته، ترسناک، زیبا — ولی انسانیت ندارد |
| Silent Protocol | Silence as choice | نبرد نهایی — یا بیدارش کن یا بگذار ساکت بمیرد |

**نجات:** اگر بعد از شکست باس احساس "وای که سخت بود" بیشتر از "وای که معنادار بود" باشد، باس شکست خورده.

## NPC

```
Every NPC loses something.
```

هیچ NPC کامل نیست. هر کدام چیزی را از دست داده‌اند. و آن فقدان، انگیزه‌شان است.

- **Keepers:** چیزی را حفظ می‌کنند (قل، خاطره، مکان)
- **Wanderers:** چیزی را جستجو می‌کنند (اما نمی‌دانند چه)
- **Broken:** چیزی را از دست داده‌اند (و دیگر همان نیستند)

| NPC | نوع | چه چیزی را از دست داده |
|----|------|----------------------|
| Engineer Kara | Keeper | مکایش (Atlas) — نمی‌داند Atlas همان Guardian است |
| Ghost Operator | Wanderer | ایستگاهش — نمی‌داند سقوط کرده |
| Last Defender | Keeper | دستور عقب‌شینی — هرگز صادر نشد |
| Gardener's Apprentice | Wanderer | آزادی‌اش — ریشه‌هایش نگهش داشته‌اند |
| Forsaken Pilot | Broken | مأموریتش — هنوز منتظر "تمام است" |
| Last Child | Broken | دنیایش — هنوز پنهان است |

**نجات:** اگر بازیکن بعد از صحبت با NPC احساس "چقدر جالب" بیشتر از "چقدر غم‌انگیز" کند، NPC شکست خورده.

## Weapons

```
Every weapon tells a story.
```

هیچ سلاحی فقط عدد نیست. هر سلاح یک تاریخچه دارد. یک صاحب قبلی. یک دلیل برای وجودش.

- پیدا کردن سلاح = پیدا کردن قصه
- ارتقای سلاح = ادامه دادن قصه
- هر سلاح ظاهر مخصوص دارد (نه فقط رنگ متفاوت)
- توضیحات سلاح لور می‌گوید، نه فقط stats

| سلاح | قصه |
|------|-----|
| Assault Rifle | سلاح استاندارد مکاهای نظامی. هزاران سال است که کار می‌کند. |
| Shotgun | ساخته‌شده برای نبرد شهرنشین. صاحبش: یک نیروی پلیس که هرگز بازنگشت. |
| Railgun | سلاح ضد-مکا. طراحی‌شده در جنگ. صاحبش: مهندسی که علیه مکاهای خودش استفاده کرد. |
| Energy Blade | ساخته‌شده از هسته Neural Overseer. هنوز گرسنه است. |
| Rocket Launcher | سلاح سنگین تخریب. صاحبش: واحدی که خودش را فدا کرد تا پل را نابود کند. |

**نجات:** اگر بازیکن سلاح را فقط به خاطر damage انتخاب کند، نه به خاطر قصه، سلاح شکست خورده.

## Music

```
Silence is part of the soundtrack.
```

موسیقی همیشه پخش نمی‌شود. سکوت مهم‌ترین ابزار صوتی است.

- **Act I:** صدای مکانیکی. تق‌تق. بخار. هیچ ملودی.
- **Act II:** باد. آب. فلز. سکوت مطلق.
- **Act III:** آژیر. ضربان. ملودی نظامی ناقص.
- **Act IV:** پرندگان. باد. آب. ملودی باستانی دور.
- **Act V:** سکوت. جرقه. تنفس.
- **Final:** یک نت. فقط یک نت.

**موسیقی فقط در لحظات مهم:** وقتی باس وارد می‌شود. وقتی پاداش پیدا می‌شود. وقتی لور کشف می‌شود. بقیه وقت: سکوت یا صدای محیط.

**نجات:** اگر بازیکن موسیقی را "نخواهد" ببندد، موفق بوده. ولی اگر بعد از ۱۰ دقیقه خسته شود، شکست.

## World

```
The world is the main character.
```

بازیکن مهم‌ترین شخصیت نیست. **دنیا** مهم‌ترین شخصیت است. بازیکن فقط مهمان است.

- هر منطقه هویت دارد (رنگ، معماری، صدا، دشمن، لور)
- هیچ منطقه فقط "سطح" نیست
- دنیا تغییر می‌کند — طبیعت بازمی‌گردد، فلز زنگ می‌زند، آب بالا می‌آید
- هر شیء در دنیا دلیل دارد (طبق قانون طلایی)
- زمان حس می‌شود — هزار سال سکوت در هر دیوار دیده می‌شود

**نجات:** اگر بازیکن screenshot بگیرد نه به خاطر "جنگ خوب بود" بلکه به خاطر "اینجا زیبا بود"، دنیا موفق بوده.
